# Arch Updater for Noctalia v5

Native Luau port of the Noctalia v4 Arch Updater.

## Features

- Arch repository updates through `checkupdates`
- AUR updates through `yay -Qua`
- Optional Flatpak updates
- Count bar widget and update-details panel
- Configurable update command

## Install

```bash
bash install.sh
```

Enable `aaronroquefonseca/arch-updater` in Noctalia Plugins, then add its `bar` widget.
